관련 독서 내용 정리

***생활코딩

-어떤 웹 페이지의 태그를 삭제하고 싶거나 어떤 태그의 자식 태그를 추가하고 싶다면 docment 객체, 이 객체에 필요한 메서드가 포함. 만약 document 객체로도 찾을 수 없다면 DOM 객체로 수색 범위를 넓힘. document 객체는 DOM의 일부이기 때문
-웹페이지가 아니라 웹 브라우저 자체를 제어해야 한다면 window 객체. 현재 열려있는 웹 페이지의 주소가 무엇인지 알아내야 할 수도 있고, 새 창을 열어야 할 수도 있고, 웹 브라우저의 화면 크기를 자바스크립트를 통해 알아야 한다면 window 객체에 속한 프로퍼티나 메서드가 도움
-웹 페이지를 리로드하지 않고도 정보를 변경하고 싶다면 Ajax 가 필요. Ajax는 현대적인 웹 앱을 만드는 데 필수적인 테크닉. 
-웹 페이지가 리로드돼도 현재 상태를 유지하고 싶다면 cookie를 배워야 함. cookie와 함께라면 사용자를 위한 개인화된 서비스를 제공할 수 있음.
-인터넷이 끊겨도 동작하는 웹 페이지를 만들고 싶다면 offline web application. 
-화상 통신 웹 앱을 만들고 싶다면 webRTC
-사용자의 음성을 인식하고 음성으로 정보를 전달하고 싶다면 speech 로 시작하는 API 를 검색
-3차원 그래픽으로 게임을 만들고 싶다면 WebGL 확인
-가상현실에 관심이 많다면 WebVR

***개발자가 되고 싶으세요? 중에서

 이미 아이폰, 안드로이드, 윈도우 폰, 블랙베리, 파이어폭스 OS에 이르는 다양한 운영체제를 지원하면서 개발자는 모든 작업을 HTML+JavaScript+CSS만을 사용해서 앱을 개발할 수 있도록 도와주는 프로젝트 들이 존재한다. 대표적으로 코르도바 Cordova 같은 프로젝트가 있다. 코르도바는 HTML5에서 제공하는 기능을 최대한 활용하면서 하드웨어를 사용하는 기능은 확장된 JavaScript를 통해 사용할 수 있다. 코르도바를 사용하면 동일한 HTML+JavaScript+CSS 소스를 가지고 다양한 플랫폼을 지원하는 앱을 쉽게 작성할 수 있을 것이다.
